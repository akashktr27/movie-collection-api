API Documentation

check the health of server

GET /register

Returns a access token for auth

GET /movies

Returns a list of movies with pagination. Requires authentication.

Get /collection

Returns a list of collection. Requires authentication.

Post /collection

Create collection and returns uuid as response. Requires authentication.

GET /collection/{uuid}

Returns details of a specific collection identified by its uuid

PUT /collection/{uuid}

Updates a specific collection identified by its uuid. where you can update movies, title and description of collection. Requires authentication.

body: { "uuid": "be8c6663-11b5-42cd-8b5b-4ab7473a1506", "title": "some title5_update_", "description": "some desc5_update", "movies": [ ] }

DELETE /collection/{uuid}

Deletes a specific collection identified by its ID. Requires authentication.

GET /request-count

Returns the number of request received by server . Requires authentication.

POST /request-count

Set the count to zero. Requires authentication.
